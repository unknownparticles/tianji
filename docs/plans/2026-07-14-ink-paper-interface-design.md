# 宣纸国风界面与解读渲染设计

## 目标

将当前偏重圆角卡片、阴影与渐变的页面调整为更克制的宣纸式国风工具界面，同时保证本地与 AI 返回的 Markdown 片段能以一致方式呈现。

## 视觉方向

页面以暖白宣纸为底、墨色为结构、朱砂红为唯一主要强调色。顶部使用印章式标题与细分隔线建立仪式感；起卦区、卦象区和解读区使用边线、留白与轻微纸纹层级区分，避免多个浮动卡片叠加。按钮、设置和历史仍保留原有入口，但采用低圆角、线性边框和清晰图标语义。

## 布局

桌面维持“起卦操作 / 卦象”双栏，栏间采用竖向墨线分隔，而不是两个独立阴影卡片。移动端自然收为单栏。铜钱场景保留现有动画与无障碍状态，但底色改为墨色留白舞台；结果区改为有卷首与落款的正文阅读面，标题、引文、强调文本都有独立的文字层级。

## 解读渲染

新增纯展示组件，将结果逐行识别为标题、引用、强调行或正文。引用内容先去掉 `> `，再解析行内 `**...**`，因此 `> **主要依据｜乾卦**` 会呈现为引用框内的加粗标题，而不是原始 Markdown 标记。未知 Markdown 保留为文本，避免把远程内容当作 HTML 注入。

## 验证

以 Vite SSR 测试验证引用内加粗、普通引用、强调行和标题都输出正确语义标签；同时检查主要界面已使用新的宣纸布局标记。完成后删除临时测试，再运行 TypeScript、生产构建和差异检查。
